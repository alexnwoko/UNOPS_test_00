# AI Tool Disclosure (Required Annex)

This submission used an AI assistant, as the assessment permits ("The use of
productivity enhancing tools such as Artificial Intelligence agents is
authorized" / "The use of AI to help you in coding and development is
encouraged"). This annex discloses where AI was used, the tool and model, the
prompts, and the sources relied upon.

## AI tool and model
- **Tool:** Anthropic Claude (Cowork desktop assistant, with a sandboxed code/execution environment).
- **Model:** Claude Opus 4.
- **How it was used:** to scaffold code, run training and tests in a sandbox, and draft documentation. All outputs were reviewed, executed, and verified by me before inclusion.

## Sources relied upon
- **Dataset:** Kaggle "Natural Language Processing with Disaster Tweets"
  (`nlp-getting-started`), ~7,613 labelled training tweets. Obtained for training
  via a public mirror of the same data
  (`https://huggingface.co/datasets/venetis/disaster_tweets`).
- **Libraries:** scikit-learn (TF-IDF, LogisticRegression, CalibratedClassifierCV),
  FastAPI + Uvicorn, pandas, NumPy, joblib.
- No external/paid LLM API is called at inference time; the model runs in-container.
